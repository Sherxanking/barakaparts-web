1. **GitHub'da repository yarating** (yoki mavjud repository'ga qo'shing)
2. **Privacy Policy faylini yarating**: `privacy-policy.md` yoki `privacy-policy.html`

   - Repository Settings ‚Üí Pages
   - Source: `main` branch, `/docs` folder
   - Save
4. **URL olasiz**: `https://yourusername.github.io/repository-name/privacy-policy.html`

```
Biz quyidagi ma'lumotlarni yig'amiz:

1. Shaxsiy Ma'lumotlar:
   - Email manzili (authentication uchun)

```
Biz yig'ilgan ma'lumotlarni quyidagi maqsadlarda ishlatamiz:

- Ilova funksiyalarini ta'minlash
- Foydalanuvchi autentifikatsiyasi
- Ehtiyot qismlari inventarini boshqarish
- Buyurtmalarni kuzatish
- Ilova xavfsizligini ta'minlash
- Foydalanuvchi yordami ko'rsatish

```
Bizning ilovamiz Supabase (backend service) ishlatadi. Ma'lumotlar quyidagi uchinchi tomonlarga uzatiladi:

- Supabase (database va authentication uchun)

  - Privacy Policy: https://supabase.com/privacy

Biz ma'lumotlarni reklama yoki marketing maqsadlarida uchinchi tomonlarga sotmaymiz.
```

```
Biz sizning ma'lumotlaringizni himoya qilish uchun quyidagi choralarni ko'ramiz:

- Barcha ma'lumotlar HTTPS (TLS/SSL) orqali shifrlangan holda uzatiladi
- Parollar xavfsiz hash qilingan holda saqlanadi
- Database'da Row Level Security (RLS) siyosatlari mavjud
- Muntazam xavfsizlik tekshiruvlari
```

### 6. Foydalanuvchi Huquqlari (User Rights)

```
Siz quyidagi huquqlarga egasiz:

- Ma'lumotlaringizni ko'rish

- Ma'lumotlaringizni eksport qilish
- Account'ni o'chirish

Agar siz ma'lumotlaringizni o'chirmoqchi bo'lsangiz, ilova ichida Settings ‚Üí Delete Account'ni tanlang yoki biz bilan bog'laning.
```

### 7. Cookie va Tracking

```
Bizning ilovamiz cookie yoki tracking texnologiyalarini ishlatmaydi. Analytics yoki reklama tracking yo'q.
```

```

```
Biz ushbu Privacy Policy'ni vaqt-vaqt bilan yangilashimiz mumkin. O'zgarishlar haqida sizni xabardor qilamiz.

## üöÄ Qadam-baqadam Yechim

### Step 1: Privacy Policy Yaratish

1. **Markdown fayl yarating**: `privacy-policy.md`
2. **Yoki HTML fayl yarating**: `privacy-policy.html`
3. **Yuqoridagi mazmunni ko'chirib qo'ying**

### Step 2: GitHub Pages'ga Joylashtirish

1. **GitHub'da repository yarating** (yoki mavjud repository'ga qo'shing)
2. **Faylni repository'ga qo'shing**:
   ```bash
   git add privacy-policy.md
   git commit -m "Add privacy policy"
   git push
   ```
3. **GitHub Pages'ni yoqing**:
   - Repository ‚Üí Settings ‚Üí Pages
   - Source: `main` branch, `/docs` folder
   - Save
4. **URL olasiz**: `https://yourusername.github.io/repository-name/privacy-policy.html`

### Step 3: Google Play Console'ga Qo'yish

1. **Play Console ‚Üí App ‚Üí Policy ‚Üí App content**
2. **Privacy Policy** bo'limiga o'ting
3. **URL kiriting**: `https://yourusername.github.io/repository-name/privacy-policy.html`
4. **Save**
